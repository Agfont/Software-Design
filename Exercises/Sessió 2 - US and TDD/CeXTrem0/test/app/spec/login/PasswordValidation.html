REQ-001 System Login

The system shall provide system logon function that will be used when a user attempts to use the system. The user needs to provide credentials in a form of username and password pair.

Examples

For the purpose of demonstration the system under test contains a system login function with username and password hardcoded to "johndoe" and "123abc!@#" respectively. All other combinations should fail.

Username Password Sucess
john doe123abc!@# no
admin admin no
johndoe 123abc!@# yes
